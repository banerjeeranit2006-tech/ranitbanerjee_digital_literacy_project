## Module Summaries

### Task 1 – Digital Literacy Infographic
Created a one-page visual infographic covering: what digital literacy is, safe internet practices, useful student tools, professional online presence, and email etiquette. Built using HTML/CSS (browser-based, no installation required).

### Task 2 – Student Digital Portfolio
Set up professional profiles on GitHub, LinkedIn, and Kaggle. Created a GitHub profile README and filled in the LinkedIn Education section with current degree details.

### Task 3 – Coding & Collaboration Platforms
Completed beginner challenges on HackerRank (Solve Me First, Python Introduction). Built a 5-question Digital Literacy Awareness Quiz using Google Forms, linked to a Google Sheets response tracker.

### Task 4 – Professional Email & Etiquette Guide
Drafted two professional emails (deadline extension request + internship inquiry). Created a Social Media Do's and Don'ts checklist with 7 Do's and 7 Don'ts for college students.

### Task 5 – Cybercrime Awareness
Wrote a 230-word case study on UPI/Online Payment Fraud. Created a 10-point prevention checklist including 2 UPI-specific tips and official reporting channels (cybercrime.gov.in & Helpline 1930).
